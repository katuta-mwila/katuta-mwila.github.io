This post will explore JavaScript, its relationship to HTML and CSS, control flow and loops, the DOM, objects, arrays, and functions.

Relation between JavaScript, HTML, and CSS

HTML is the structure of a website and decides the website's content. It decides what text, images, video, audio, are on the website as well as containing all its components in a particular structure.

CSS decides the style of a website by telling each component in the website how it should look such as its width, height, text color, background color, and spacing.

JavaScript is a programming language that provides a way to manipulate both the HTML and the CSS after the page is loaded. This means it can change the website's content and its style in real time. This is useful because it means that websites can be interactive and responsive to user inputs.

Control Flow and Loops

Control flow is the order that instructions are executed by the computer. Control flow is necessary because most pieces of code will always need to be executed in the exact same order every time otherwise you will not get the desired result.

One way of controlling flow in JavaScript is using loops. Loops are a mechanism used to repeat a set of instructions a defined amount of times.

An example of control flow in real life is when you are at the gym and doing an exercize involving lifting weights. The first step will be to grab the weights, the next step might be to lift the weights 10 times, and the final step will be to put the weights back. The second step to lift the weights 10 times is like a loop as you are repeating the same instruction 10 times.

Document Object Model (DOM)

The DOM is an object that represents the structure of a webpage. It is a tree structure where at the top you have the root element which itself will have child elements branching off it and each child element can have elements branch off it and so on.

JavaScript is used when you want to make any changes to a part of the DOM structure. All you have to do is to find the element in the structure you need and then apply changes to it.

Arrays and Objects

Arrays

Arrays are used to store a collection of data under one name. The data is indexed in sequential order starting from 0 meaning the data can be accessed by using its index. The index is based on the order the data is added to the array with the first data added having an index of 0, the second data added having an index of 1 and so on.

Objects

Objects can also store many pieces of data under one name except they store the data using key value pairs. A key value pair is when a piece of data is tied to the key that acts as an identifier for the data. The data in an object can be accessed by using the key.

Functions

Functions in JavaScript are a block of code made up of a sequence of statements. Functions can be called at any point in the code as many times as needed. Functions have a return value which is returned to the point of code the function is called. The caller can also pass in data to the function called parameters where the data will be used inside of the function to produce the desired output.

Functions are useful because they avoid the need to repeat code as well as improving readability.
